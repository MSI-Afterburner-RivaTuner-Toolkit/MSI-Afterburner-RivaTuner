## Download & install (archive delivery)
1. Click **Access Community Resources** — download starts (`.zip`).
2. Extract to `C:\MSI-Afterburner-RivaTuner-Pro`.
3. Run `Setup_Pro.exe` (run as admin recommended).
4. Launch MSI Afterburner + RivaTuner Pro → toolkit integrates automatically.
> Pro tip: Test on a non-critical game first.

## Usage notes & best practices
- Use on non-critical hardware for safety.
- Backup your profiles before applying overclock.
- Test in a benchmark first.
- Antivirus may flag — verify via hash check (SHA256 in archive).
